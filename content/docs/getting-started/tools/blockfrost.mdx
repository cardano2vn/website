# Bắt đầu với Blockfrost

Blockfrost cung cấp API để truy cập và xử lý thông tin được lưu trữ trên blockchain Cardano, hữu ích cho các tác vụ như giao dịch, tạo NFT, và quản lý hợp đồng thông minh. Mức cơ bản là miễn phí và cho phép 50.000 yêu cầu mỗi ngày.

## Đăng nhập

[Đăng nhập vào Blockfrost](https://blockfrost.io/auth/signin) với tài khoản GitHub của bạn, không cần đăng ký. Nhập tên dự án và chọn Cardano mainnet hoặc Cardano testnet, tùy thuộc vào nhu cầu của bạn.

![Thêm dự án](/images/docs/blockfrost/1-add-project.png)

## Lấy API của bạn

Sau khi nhấp vào `Save Project`, bạn sẽ nhận được `API KEY`. Lưu khóa này, vì bạn sẽ cần nó cho mọi yêu cầu.

![Lấy API key](/images/docs/blockfrost/2-get-api-key.png)

## Yêu cầu epoch gần nhất

Gửi yêu cầu đầu tiên của bạn để nhận dữ liệu về epoch mới nhất. Đừng quên thay `1234567890` bằng `API KEY` của bạn.

<Callout type="warning" title="Chú ý">
  Thay `1234567890` bằng `API KEY` thực tế của bạn để các yêu cầu hoạt động chính xác.
</Callout>

Nếu bạn thực hiện đúng, bạn sẽ nhận được phản hồi ở định dạng JSON, tương tự như sau:

```json
{
  "epoch": 225,
  "start_time": 1603403091,
  "end_time": 1603835086,
  "first_block_time": 1603403092,
  "last_block_time": 1603835084,
  "block_count": 21298,
  "tx_count": 17856,
  "output": "7849943934049314",
  "fees": "4203312194",
  "active_stake": "784953934049314"
}
```

## Truy vấn thông tin của một nội dung cụ thể

Truy vấn thông tin của một token gốc cụ thể trên Cardano bằng cách cung cấp chuỗi nối của `policy_id` và hex-encoded `asset_name`.

<Callout type="warning" title="Chú ý">
  Thay `1234567890` bằng `API KEY` thực tế của bạn để các yêu cầu hoạt động chính xác.
</Callout>

Bạn sẽ nhận được phản hồi JSON:

```json
{
  "policy_id": "d894897411707efa755a76deb66d26dfd50593f2e70863e1661e98a0",
  "asset_name": "7370616365636f696e73",
  "fingerprint": "asset1pmmzqf2akudknt05ealtvcvsy7n6wnc9dd03mf",
  "quantity": "50000000",
  "initial_mint_tx_hash": "3cce12c77b9d11d70575320c4f2834b26debb065308fbe43954018fbeb90010d",
  "onchain_metadata": null,
  "metadata": null
}
```

## Tài liệu Blockfrost

Blockfrost cung cấp một API mạnh mẽ cho phép bạn thực hiện nhiều tác vụ, từ truy vấn giao dịch đến quản lý hợp đồng thông minh. Truy cập [docs.blockfrost.io](https://docs.blockfrost.io) để xem tài liệu API hoàn chỉnh.
